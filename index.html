Project Description

Birzeit Flat Rent is a comprehensive flat rental management system that allows customers to search for and rent flats, owners to list their properties, and managers to oversee the entire process.

Main Features:

  • Advanced flat search with multiple filters
  • User registration system (Customers, Owners, Managers)
  • Flat listing and management
  • Rental booking system
  • Preview appointment scheduling
  • Message system for notifications
  • Role-based access control
